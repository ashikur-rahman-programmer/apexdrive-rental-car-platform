# ApexDrive - Car Rental Platform

Live Site: https://apexdrive-rental-car.vercel.app

---

## Overview

ApexDrive is a full-stack Car Rental Platform that allows users to explore available cars, view detailed information, rent vehicles, and manage their bookings. Users can also add, update, and delete car listings with secure authentication and protected routes.

---

## Key Features

- Secure authentication system with JWT-based authorization
- Full CRUD functionality for car listings (Add, Update, Delete)
- Complete booking system for renting cars with user-specific booking history
- Search and filter cars by name and type using MongoDB queries
- User profile management with personalized data access
- Fully responsive design for mobile, tablet, and desktop devices
- Protected private routes with session persistence using HTTPOnly cookies

---

## Tech Stack

- Frontend: React / Next.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: Better Auth, JWT, HTTPOnly Cookies
- Deployment: Vercel (Client)

---

## Core Functionalities

- Explore all available cars in a modern grid layout
- View detailed car information before booking
- Book cars with custom booking options
- Manage personal bookings and added cars
- Update or delete car listings securely
- Persistent login system without session loss on refresh
